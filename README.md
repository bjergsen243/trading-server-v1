Trading Server


## Description

A trading server with NestJs, MongoDb and Redis


## Installation

Install Node20, docker, docker-compose

```bash
npm install
```

## Config & Pre-Run

Create `.env` from `.env.example`.
## Docker

``bash
docker compose -f docker-compose.yml up -d
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
